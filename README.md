Exclusive Gifts Shop - Front-End E-Commerce Interface
A client-side website for an online store selling exclusive gifts. 
The project implements a fully functional user interface that integrates with the existing e-commerce API.

## Description
MODO Exclusive Gifts Shop ‚Äì Front‚ÄëEnd E‚ÄëCommerce Interface. This is the frontend of an online store selling exclusive gifts. It is a responsive web site developed using HTML, CSS, and JavaScript, including:
- Home page with 
   - interactive banner carousel showing the 3 latest products;
   - catalog with categories;
   - esponsive thumbnail grid;
   - search bar;
   - subscription form for updates
- Product page with
   - product image with all the necessary information, customer reviews;
   - share button;
   - "Add to cart" button accessible only to logged-in users;
   - section with some randomly added products
- Cart page with
   - list of products in the cart and the total price;
   - clear cart function;
   - button to proceed to checkout;
- Checkout page with
   - form with various payment methods
   - contact information form
   - duplication of delivery address from contact information form
   - summing up the delivery price with the order price
   - list of products in the cart and the total price;
   - "Pay order" button which redirect to Success page and clear cart
- Success page with
   - message of confirmation of order placement and duplication of delivery address from contact information form
- Login page with
   - validated login form that requests and saves a token to the browser
- Register page with
   - validated register form to create a new account (email and password)
- Coming soon page
